# 📖 Horror Wiki

**Horror Wiki** adds a mysterious in-game book that documents and explains every horror entity in your world.

From SCPs to unknown creatures and classic monsters, this book serves as your ultimate survival guide.

---

## 🧠 Features

- 📚 **In-Game Encyclopedia**  
  A detailed book that explains all horror entities you encounter.

- 👁️ **Entity Documentation**  
  Learn about:
  - Origins  
  - Behavior  
  - Abilities  
  - Weaknesses  

- ⚠️ **Survival Tips**  
  Get useful advice on how to survive encounters with each entity.

- 🌌 **Cross-Universe Content**  
  Includes creatures from:
  - SCP Foundation  
  - Internet horror  
  - Custom monsters  

---

## 🔍 How it works

1. Checks what Horror Mods you're using.
2. Provides you Informations about the entities (if compatible)
